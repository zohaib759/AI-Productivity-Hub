# 🚀 AI Productivity Hub

An AI-ready productivity web application built with **Flask** that helps users organize their daily workflow through task management, note-taking, dashboard analytics, and user profile management. The project is designed with a modular architecture and will be expanded with AI-powered productivity tools.

## 📖 Overview

AI Productivity Hub is a full-stack Flask application that allows users to:

- ✅ Create and manage personal notes
- ✅ Organize daily tasks
- ✅ Track completed and pending work
- ✅ View productivity statistics
- ✅ Manage personal profiles
- ✅ Secure authentication system

The project follows a scalable Flask Blueprint architecture and is being developed into an AI-powered productivity platform.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/zohaib759/AI-Productivity-Hub.git

cd AI-Productivity-Hub
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv

venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Database Migrations

```bash
flask db upgrade
```

---

## Start Server

```bash
python run.py
```

or

```bash
flask run
```
